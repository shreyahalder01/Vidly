# Vidly - Video Downloader

Vidly is a web application that lets you download videos from a given URL.

## Features
- Choose video quality before downloading
- Real-time download progress bar
- Minimal and modern UI

## Requirements
- Python 3.8+
- Flask
- yt-dlp
